## Description

# [Pi Coding Agent (Free Course)]

## 1. 핵심 주제
Pi Coding Agent는 오픈 소스 기반의 에이전틱 코딩 도구로, 사용자에게 다양한 AI 모델과의 통합 및 확장 기능을 제공합니다.

## 2. 주요 개념 정리

- **Pi Coding Agent 정의**: 에이전틱 코딩 도구로, 다양한 AI 모델을 통합하고 확장할 수 있는 오픈 소스 소프트웨어.
- **작동 원리**:
  - 최소한의 기본 도구 제공: read, bash, edit, write.
  - 커스터마이즈 가능한 시스템 프롬프트와 확장 기능.
- **중요한 주장**: 사용자 맞춤형 AI 코딩 환경 구축이 가능하며, 다른 AI 코딩 도구와 비교 시 유연성과 확장성이 뛰어남.
- **근거**: 다양한 API 및 모델과의 통합, 사용자 정의 확장 기능 및 시스템 프롬프트 가능.
- **시스템 프롬프트**: 최소한의 기본 상태에서 사용자가 원하는 지침 추가 가능.
- **확장 기능**: TypeScript를 사용하여 사용자 맞춤형 기능 추가 가능.

## 3. 세부 설명

1. **설치 및 시작**:
   - 설치: pi.dev에서 설치 명령어 복사 후 터미널에 붙여넣어 설치.
   - 시작: 설치 후 'pi' 명령어로 에이전트 시작.

2. **모델 선택**:
   - 다양한 AI 모델 제공자 선택 가능 (예: OpenAI, Google Cloud, GitHub Co-Pilot).
   - Open Router를 통해 모든 AI 모델에 접근 가능, 이때 API 키와 크레딧 필요.

3. **기능 설정**:
   - 커맨드(/command) 및 핫키를 통해 다양한 기능 설정 가능.
   - 세션 관리: 세션 내보내기/가져오기, 포크(fork) 기능 활용.

4. **확장 기능**:
   - TypeScript 파일로 확장 기능 추가 가능.
   - UI 커스터마이징 및 워크플로우 구축 가능.

## 4. 예시 / 적용

- **확장 기능 구현 예시**: Git 상태를 UI에 표시하는 간단한 UI 확장 기능 개발.
- **시스템 프롬프트 커스터마이제이션**: 코딩 외 다른 작업을 위한 에이전트로 변환.

## 5. 시사점

- Pi Coding Agent는 사용자의 필요에 따라 강력히 커스터마이즈 및 확장 가능하므로, AI 코딩 워크플로우에 혁신적 도구로 자리잡을 수 있음.
- 비용 효율적인 모델 선택 및 사용에 대한 전략적 접근 필요.

## 핵심 정리

- **Pi Coding Agent**는 오픈 소스 기반의 에이전틱 코딩 도구이다.
- **AI 모델 통합** 및 **확장 기능**이 가능하다.
- **시스템 프롬프트**와 **확장 기능**은 사용자 정의가 가능하다.
- **Open Router**를 통해 다양한 AI 모델에 접근할 수 있다.
- **세션 관리** 기능을 통해 작업 효율성을 높인다.
- **TypeScript**로 확장 기능을 손쉽게 추가할 수 있다.
- **비용 절감**을 위해 모델 선택에 신중해야 한다.

4. **Open Router 설정**: Open Router를 통해 다양한 AI 모델에 접근할 수 있으며, 이를 위해 API 키와 크레딧이 필요합니다.
