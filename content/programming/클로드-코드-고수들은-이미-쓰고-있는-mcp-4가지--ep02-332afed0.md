# 클로드 코드 고수들은 이미 쓰고 있는 MCP 4가지 | EP.02



## 1. 핵심 주제

- **MCP(Multi-Channel Protocol)의 이해와 활용법**: 클로드 코드를 확장하고 다양한 외부 도구와 연결하여 더 강력한 기능을 발휘할 수 있도록 돕는 MCP의 작동 원리와 사용법을 이해한다.



## 2. 주요 개념 정리



1. **MCP의 정의**

   - **MCP**: Multi-Channel Protocol의 약자로, 클로드 코드가 외부 도구와 연결될 수 있도록 하는 표준 규격.

   - 스마트폰의 앱 설치와 유사하게, 클로드 코드에 MCP를 설치하여 다양한 기능을 확장함.



2. **MCP 작동 원리**

   - 클로드 코드에서 프롬프트 입력 시, MCP 서버가 외부 도구와의 연결을 통해 자동으로 작업을 수행.

   - 작업 과정은 사용자가 직접 알 필요 없이 자동화되어 처리됨.



3. **중요한 주장**

   - MCP를 통해 클로드 코드의 활용 범위를 확장할 수 있으며, 비개발자도 쉽게 사용할 수 있다.

   - MCP는 다양한 상황에서 자동화된 솔루션을 제공하여 작업 효율성을 높인다.



4. **주요 MCP 소개**

   - **플레이라 MCP**: 웹사이트 탐색 및 상호작용 자동화.

   - **컨텍스트 7 MCP**: 최신 문서와의 연동을 통해 클로드 코드의 최신성을 유지.

   - **파이어크롤 MCP**: 대량의 웹 데이터를 빠르게 수집하고 정리.

   - **시퀀셜 씽킹 MCP**: 복잡한 문제 해결을 위한 단계적 사고 과정 지원.



## 3. 세부 설명



1. **MCP 설치 및 작동**

   - 설치는 간단하며, 터미널이나 명령어를 통해 쉽게 설치 가능.

   - 설치 후 클로드 코드가 자동으로 MCP를 활용하여 작업 수행.

   - 특정 MCP는 API 키가 필요할 수 있으며, 자동화 도구를 통해 쉽게 설정 가능.



2. **주요 MCP 작동 방식**

   - **플레이라 MCP**: 웹사이트에서 직접 클릭 및 데이터 입력을 자동화, 사용자가 요청하면 브라우저 제어.

   - **컨텍스트 7 MCP**: 최신 공식 문서를 실시간으로 가져와 클로드 코드의 코드 작성 오류 방지.

   - **파이어크롤 MCP**: 웹사이트 전체 크롤링, JSON이나 CSV 등으로 데이터를 정리 및 분석.

   - **시퀀셜 씽킹 MCP**: 복잡한 문제를 단계별로 사고하여 분석 및 해결 지원.



## 4. 예시 / 적용



1. **플레이라 MCP 적용 예시**

   - 강남역 맛집 검색 후 구글 시트에 자동으로 추가.

   - 웹사이트 로그인 및 데이터 입력 자동화.



2. **컨텍스트 7 MCP**

   - 최신 문법을 이용한 코드 작성 시, 실시간 공식 문서 참조로 오류 최소화.



3. **파이어크롤 MCP**

   - 토스테크 최신 아티클 크롤링 및 CSV 파일 생성.

   - 무신사 랭킹 상품 데이터 수집 및 정리.



4. **시퀀셜 씽킹 MCP**

   - 웹사이트 개발 시 단계별 사고를 통해 문제 해결.

   - 경쟁사 분석을 위한 단계적 데이터 수집 및 분석.



## 5. 시사점



- MCP의 활용은 클로드 코드의 기능을 대폭 확장하여 비개발자도 복잡한 작업을 쉽게 수행할 수 있게 한다.

- 자동화된 데이터 수집과 분석은 효율성을 높이고, 다양한 분야에서의 활용 가능성을 제시한다.

- 복잡한 문제 해결을 위한 단계적 사고 지원은 AI 활용의 새로운 지평을 열어준다.

- **주요 MCP**: 플레이라, 컨텍스트 7, 파이어크롤, 시퀀셜 씽킹.

- **컨텍스트 7 MCP**: 최신 문서 연동.

- **파이어크롤 MCP**: 대량 데이터 크롤링.

- **시퀀셜 씽킹 MCP**: 단계적 문제 해결.

- **효율성과 확장성**: 다양한 외부 도구와의 연결을 통한 클로드 코드의 기능 확장.
