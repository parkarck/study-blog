## 2. 주요 개념 정리

- **API 정의**: Application Programming Interface의 약자로, 소프트웨어 간의 상호작용을 가능하게 하는 인터페이스.

- **Qwen 3.5의 역할**: 데이터 과학 작업을 용이하게 하여, 비전문가도 데이터 분석을 수행할 수 있도록 돕습니다.

- **무료 접근성**: 누구나 자유롭게 사용할 수 있어 데이터 과학의 진입 장벽을 낮춥니다.



## 3. 세부 설명

### API 작동 원리

1. **요청**: 사용자가 특정 데이터를 요청합니다.

2. **처리**: API가 요청을 받아 서버에서 데이터를 처리합니다.

3. **응답**: 처리된 데이터를 사용자에게 반환합니다.



### Qwen 3.5의 기능

- **데이터 수집**: 다양한 소스에서 데이터를 자동으로 수집.

- **데이터 전처리**: 노이즈 제거, 형식 변환 등 데이터 클린징 작업 지원.

- **데이터 분석**: 통계 분석, 머신러닝 모델 적용 및 결과 시각화.
