# [Study Clipper] Claude Code 세팅: 5분 만에 프로급으로



## 📋 요약



이 영상은 Claude Code를 효율적으로 활용하기 위한 세 가지 핵심 요소(CLAUDE.md, 에이전트, 훅)와 5분 만에 프로급 세팅을 완료할 수 있는 "Claude Forge" 도구를 소개합니다.



### 🔑 핵심 개념



1. **CLAUDE.md**: Claude Code에게 주는 업무 매뉴얼

   - 회사의 신입 사원에게 업무 매뉴얼을 주는 것과 동일

   - 잘 작성하면 답변 품질이 완전히 달라짐



2. **에이전트 시스템**: 11명의 전문가 고용

   - 기획자, 설계자, 테스트 전문가, 코드 리뷰어, 보안 전문가 등

   - 각 에이전트가 독립적으로 동시 작업 가능



3. **훅(Hooks)**: 자동화 장치

   - 아파트 수도관 누수 검사처럼 문제가 생기기 전에 자동 감지

   - 예: API 키 코드에 넣고 커밋하려고 하면 차단



### 🛠️ Claude Forge (밀키트)



- **설치 시간**: 5분

- **포함 내용**:

  - 11개 전문 에이전트

  - 36개 슬래시 명령어

  - 15개 스킬

  - 14개 보안 훅



**GitHub**: https://github.com/sangrokjung/claude-forge



### 📝 워크플로우 파이프라인



1. **Plan** → 기획자 에이전트가 프로젝트 계획 세움

2. **TDD** → 테스트 먼저 작성 (실패 방지)

3. **Review** → 코드 리뷰어가 코드 검토

4. **Handoff** → 독립적으로 검증

5. **Commit/Push/PR** → GitHub에 자동 업로드



### 🔒 보안 자동화



- API 키 코드에 넣으려고 하면 훅이 감지해 차단

- 6단계 보안 파이프라인 자동 실행

- 대기업 보안 사고 예방



### 🎯 DOE 프레임워크



1. **Directives** - 방향 제시

2. **Orchestration** - 조율

3. **Execution** - 실행
