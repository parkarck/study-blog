# 페르소나 추출 방법

## 1. 핵심 주제
카카오톡 오픈채팅 데이터를 활용하여 참여자의 페르소나를 자동으로 생성하는 방법론을 설명한다. 이 과정은 CSV 파일 형식의 채팅 데이터를 수집하여 다양한 분석 과정을 통해 개인의 대화 패턴과 전문성을 파악하고, 이를 마크다운 문서로 정리한다.

## 2. 주요 개념 정리

- **페르소나(persona)**: 특정 사용자의 특징과 성향을 이해할 수 있도록 가상의 인물 프로필을 작성하는 것.
- **데이터 수집**: 카카오톡 채팅 데이터를 CSV 파일 형식으로 수집.
- **분석 관점**: 전문 분야, 대화 참여 패턴, 주제별 입장, 영향력, 시간에 따른 변화, 커뮤니케이션 스타일.
- **페르소나 마크다운**: 분석 결과를 기반으로 작성된 사용자 프로필 문서.

## 3. 세부 설명

### 3.1 데이터 수집 및 준비
- **CSV 파일 수집**: 채팅 데이터를 CSV 형식으로 수집하여, `Date`, `User`, `Message` 컬럼을 포함한다.
- **Python 환경**: Python 3.11 이상의 환경에서 스크립트를 실행하여 데이터 수집 및 분석을 수행한다.

### 3.2 데이터 현황 및 분석
- **현황 확인**: 전체 데이터의 상태를 점검하거나, 상위 30명의 데이터를 우선적으로 확인 가능.
- **재료 추출**: SQLite를 사용하여 월별 균등 샘플링된 메시지를 추출한다. 이는 특정 사용자나 상위 사용자에 대한 데이터를 분석할 때 활용된다.

### 3.3 페르소나 작성
- **6가지 분석 관점**:
  1. **전문 분야**: 자주 언급하거나 권위 있는 답변을 제공하는 주제.
  2. **대화 참여 패턴**: 질문과 답변의 비율, 대화 주도 성향.
  3. **주제별 입장**: 기술이나 방법론에 대한 개인의 입장.
  4. **영향력**: 다른 참여자에게 미치는 추천 및 조언의 영향력.
  5. **시간에 따른 변화**: 초기와 최근의 관심사 및 관점 변화.
  6. **커뮤니케이션 스타일**: 말투, 메시지 길이, 이모지 사용 등.

### 3.4 페르소나 문서 생성
- **마크다운 형식**: 각 사용자의 페르소나를 `data/personas/{닉네임}.md`에 저장.
- **구성 요소**: 기본 정보, 전문 분야, 커뮤니케이션 스타일, 대화 참여 패턴, 주제별 입장, 시간에 따른 변화, 영향력 및 추천 패턴, 대표 발언.

## 4. 예시 / 적용
- **실제 채팅에서 추출한 대표 인용문**: 분석된 데이터를 통해 각 사용자의 대표적인 발언을 수집하고, 이를 통해 사용자 이해도를 높인다.

## 5. 시사점
- **데이터 기반 인사이트**: 페르소나 분석은 사용자 행동 및 성향을 데이터 기반으로 이해할 수 있는 도구로 활용된다.
- **적용 가능성**: 다양한 분야에서 고객 이해, 마케팅 전략 수립 등에 활용 가능.

## 핵심 정리

- 페르소나는 사용자의 대화 데이터를 기반으로 성향과 전문성을 분석한 가상 프로필이다.
- 데이터를 CSV 형식으로 수집하여 Python 환경에서 분석한다.
- 분석은 전문 분야, 대화 패턴, 주제별 입장 등 6가지 관점으로 이루어진다.
- 결과는 마크다운 문서로 정리되어 사용자 이해를 돕는다.
- 페르소나 분석은 데이터 기반 인사이트를 제공하며, 다양한 분야에 응용 가능하다.
