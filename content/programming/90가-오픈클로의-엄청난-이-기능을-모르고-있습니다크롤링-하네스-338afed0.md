# 오픈클로의 강력한 기능 및 활용법



## 1. 핵심 주제

오픈클로(OpenClo)의 최신 업데이트와 활용 가능한 기능들, 특히 크롤링 및 하네스(Harness) 기능을 중심으로 설명합니다.



## 2. 주요 개념 정리



### 오픈클로(OpenClo)

- **정의**: 다양한 작업 자동화를 돕는 오픈 소스 소프트웨어 플랫폼.

- **주요 기능**: 크롤링, 스킬 설치, 에이전트 호출 등.



### 크롤링(Crawling)

- **정의**: 웹사이트에서 데이터를 자동으로 수집하는 기술.

- **작동 원리**: 브라우저를 자동으로 제어하여 필요한 데이터를 수집.



### 하네스(Harness)

- **정의**: 목표 도달을 위해 반복적으로 에이전트를 훈련시키는 개념.

- **작동 원리**: 목표 설정 후 반복적 실행을 통해 작업을 최적화.



### ACP 채널 바인딩

- **정의**: 다양한 플랫폼 간 세션을 유지한 채로 이동할 수 있도록 돕는 기능.

- **주요 주장**: 코딩 중 세션을 유지하며 디스코드, 아이메시지 등으로 이동 가능.



## 3. 세부 설명



### 오픈클로의 주요 업데이트 및 기능

- **릴레이 기능 제거**: 기존 릴레이 기능이 사라지고 크롬 데브 MCP를 사용하는 구조로 변경됨.

- **클로허브 강화**: 클로허브를 통해 스킬 설치가 용이해짐.

- **보안 및 UI 변화**: 보안 강화 및 사용자 인터페이스(UI) 개선.



### 크롬 데브 MCP 활용법

- **설치 및 설정**: 크롬 인스펙트를 통해 리모트 디버깅 활성화 후, MCP 서버를 추가하여 사용.

- **자동화 적용**: 웹 브라우저 제어를 통해 자동으로 데이터를 수집하고 정리.



### 하네스의 구현 및 장점

- **작업의 자동화**: 설정된 목표에 따라 에이전트를 반복적으로 훈련시켜 자동화.

- **워크플로우와의 차이**: 워크플로우보다 더 동적이며 오류 시 복구 가능.



## 4. 예시 / 적용



- **경매 사이트 크롤링**: 대전 지역 경매 아파트 정보를 자동으로 수집하여 엑셀 정리.

- **API 활용**: 다양한 API 키를 통해 저렴하게 서비스를 이용하며 작업 자동화.



## 5. 시사점



- **효율적 데이터 수집**: 자동화를 통해 시간과 노력을 절약.

- **확장 가능한 구조**: 다양한 플랫폼과의 연계를 통해 무한한 가능성 제공.

- **보안과 안정성**: 보안이 강화된 환경에서 안정적인 서비스 제공.



## 핵심 정리



- 오픈클로는 작업 자동화를 지원하는 강력한 플랫폼이다.

- 크롤링을 통해 웹 데이터를 자동으로 수집하고 활용할 수 있다.

- 하네스는 목표 달성을 위한 반복적 훈련을 통해 작업을 최적화한다.

- ACP 채널 바인딩은 다양한 플랫폼 간 원활한 세션 이동을 지원한다.

- 최신 업데이트로 보안 강화 및 UI 개선이 이루어졌다.

- 크롬 데브 MCP를 이용해 자동화된 웹 데이터 수집이 가능하다.
