# [Harness-100 실전 순례 | 100개 하네스 중 내게 맞는 걸 찾는 법]



## 1. 핵심 주제

Harness-100 프로젝트에서는 100개의 다양한 하네스를 통해 유용한 에이전트 팀을 구성하여 작업 효율을 극대화할 수 있습니다. 각 하네스는 특정 작업을 자동화하고 최적화하는 데 도움을 주며, 개인의 필요에 맞춰 선택하고 수정할 수 있습니다.



## 2. 주요 개념 정리



### 하네스(Harness)

- **정의**: 특정 작업을 효율적으로 수행하기 위해 설계된 에이전트 팀의 집합.

- **구조**: 3계층 스킬 시스템으로 구성됨.

  - **Orchestrator**: 전체 워크플로우를 조율.

  - **Agent Skills**: 전문 지식을 제공.

  - **External Skills**: 외부 도구와 연동.



### 설치 및 사용

- **간단한 설치**: 저장소 클론 후 하네스 폴더 복사.

- **한국어 지원**: ko 폴더 참조.



## 3. 세부 설명



### 하네스의 구조와 작동 원리

- **3계층 구조**: Orchestrator는 워크플로우를 조정하고, Agent Skills는 특정 도메인 지식을 활용하며, External Skills는 외부 리소스를 통합.

- **복사 및 실행**: 프로젝트에 .claude 폴더를 복사해 손쉽게 사용 가능.



### 하네스의 활용

1. **YouTube Production**: 유튜브 영상 제작.

   - 팀 구성: 콘텐츠 전략가, 대본 작성가, 썸네일 디자이너 등.

   - 결과물: 전략 브리프, 대본, SEO 패키지 등.

   

2. **Code Reviewer**: 코드 품질 검사.

   - 팀 구성: 아키텍처, 보안, 퍼포먼스, QA 리뷰어.

   - 결과물: 보안 취약점, 성능 개선 포인트, 아키텍처 제안.



3. **Blog Writing**: 블로그 포스팅 작성.

   - 팀 구성: 리서처, 라이터, SEO 에디터, 리뷰어.

   - 결과물: 초안, 키워드 최적화, 품질 검증.



4. **Fullstack Webapp**: 풀스택 앱 개발.

   - 팀 구성: 프론트엔드 아키텍트, 백엔드 아키텍트 등.

   - 결과물: 프로젝트 구조, API 설계, 컴포넌트 생성.



5. **Startup Launcher**: 스타트업 런칭.

   - 팀 구성: 시장 분석가, 제품 전략가, 마케팅 전략가 등.

   - 결과물: 시장 분석, 경쟁사 벤치마크, 가격 책정 전략.



## 4. 예시 / 적용



### 커스텀 하네스 제작

- **기존 하네스 수정**: CLAUDE.md와 agents, skills 폴더 수정으로 맞춤형 하네스 제작.

- **응용**: YouTube Production에 한국어 콘텐츠 추가, 특정 프레임워크 규칙 적용.



## 5. 시사점

- **효율성 증가**: 하네스를 활용하면 수작업 대비 작업 시간을 크게 단축 가능.

- **맞춤형 에이전트 팀**: 필요에 따라 하네스를 선택 및 수정하여 최적의 팀 구성.

- **공유와 협업**: PR을 통해 다른 사용자와 맞춤형 하네스를 공유 가능.



## 핵심 정리

- **구조**: Orchestrator, Agent Skills, External Skills의 3계층.

- **설치와 사용**: 간단한 클론 및 복사로 실행.

- **대표 하네스**: YouTube Production, Code Reviewer, Blog Writing, Fullstack Webapp, Startup Launcher.

- **커스터마이징**: 필요에 따라 하네스 수정 가능.
