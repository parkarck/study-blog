## 1. 핵심 주제

Windows 운영체제의 기본 설정을 최적화하여 보안을 강화하고 불필요한 광고를 차단하는 방법.



## 2. 주요 개념 정리



### 2.1 CMD(Command Prompt) 사용

- **정의**: Windows의 명령어 기반 인터페이스로, 다양한 시스템 명령어를 실행할 수 있음.

- **작동 원리**: CMD에서 특정 명령어를 입력하여 시스템 설정 및 조정을 수행.



### 2.2 Windows 업데이트 및 보안

- **정의**: Windows 운영체제를 최신 버전으로 유지하고, 시스템 보안을 강화하기 위한 방법.

- **작동 원리**: 명령어를 통해 시스템 내 모든 앱을 최신 버전으로 업데이트하고, 보안 프로그램을 통해 악성 코드를 제거.



### 2.3 광고 차단 설정

- **정의**: Windows 시작 메뉴와 기타 시스템 영역에서 나타나는 광고를 비활성화하는 방법.

- **작동 원리**: 시스템 설정을 조정하여 광고 노출을 제한함.



## 3. 세부 설명



### 3.1 CMD를 통한 시스템 최적화

1. **CMD 실행**:

   - 검색창에 'CMD' 입력 후, 관리자 권한으로 실행.

2. **앱 업데이트**:

   - 'winget upgrade' 명령어 입력.

   - 다시 '-all' 옵션을 사용하여 모든 앱을 최신 버전으로 업데이트.



### 3.2 보안 강화

1. **MRT 실행**:

   - CMD에서 'MRT' 입력 후 실행.

   - Windows 보안 프로그램이 활성화되어 시스템 내 악성 코드를 제거.

   - 백신 프로그램의 필요성을 감소시킴.



### 3.3 광고 차단

1. **시작 메뉴 광고 제거**:

   - '시작' 검색 후 핀 바로 가기 끄기.

2. **개인 정보 보호**:

   - 개인 정보 설정에서 첫 번째와 네 번째 옵션 조정.

   - 개인 정보 수집을 제한하여 타겟 광고 차단.

## 5. 시사점



- **보안 강화의 중요성**: 운영체제의 기본 보안 설정을 강화함으로써, 악성 코드로부터 사용자 데이터를 보호할 수 있음.

- **사용자 경험 개선**: 불필요한 광고를 제거하여 보다 쾌적한 사용 환경을 제공.
