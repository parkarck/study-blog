# Mofit iOS App

## 1. 핵심 주제
Mofit iOS 앱은 iOS 17.0을 기반으로 한 피트니스 추적 애플리케이션으로, 최신 기술 스택을 활용하여 사용자의 운동 자세를 추적하고 피드백을 제공합니다.

## 2. 주요 개념 정리
- **프로젝트 개요**: iOS 17.0 기반 피트니스 추적 앱
- **기술 스택**:
  - **언어**: Swift 5.9
  - **프레임워크**: SwiftUI, SwiftData, AVFoundation, Vision
  - **플랫폼**: iOS (iPad 포함)
  - **Xcode 버전**: 15.0
  - **배포 타겟**: iOS 17.0+
- **주요 기능**:
  - 운동 자세 추적
  - 카메라 사용
  - 다크 모드 UI
  - 포트레이트 모드 전용
- **앱 정보**:
  - **Bundle ID**: com.mofit.app
  - **번들 ID 프리픽스**: com.mofit
  - **카메라 사용 설명**: "운동 자세를 추적하기 위해 카메라가 필요합니다"
- **GitHub 링크**: [Mofit iOS Repository](https://github.com/vibemafiaclub/mofit-ios)

## 3. 세부 설명

### 정의
Mofit iOS 앱은 운동 자세를 추적하여 피드백을 제공하는 피트니스 애플리케이션입니다. iOS 17.0을 기반으로 하며, Swift 5.9 언어와 다양한 프레임워크를 사용하여 개발되었습니다.

### 작동 원리
- **운동 자세 추적**: Apple의 Vision 프레임워크를 사용하여 사용자의 운동 자세를 실시간으로 분석합니다.
- **카메라 사용**: AVFoundation을 통해 카메라를 활용하여 사용자의 운동 모습을 캡처하고 추적합니다.
- **UI 구현**: SwiftUI를 사용하여 사용자가 쉽게 상호작용할 수 있는 다크 모드 UI를 제공합니다.
- **포트레이트 모드**: iPad를 포함한 다양한 iOS 장치에서 포트레이트 모드로 작동합니다.

### 중요한 주장
Mofit 앱은 최신 iOS 기술을 활용하여 사용자에게 정확한 운동 자세 분석과 피드백을 제공함으로써, 피트니스 경험을 향상시킵니다.

### 근거
- 최신 iOS 17.0 플랫폼과 Xcode 15.0을 사용하여 안정적이고 최적화된 성능 제공
- Vision과 AVFoundation 프레임워크의 활용으로 정확한 자세 분석 가능

## 4. 예시 / 적용
- **운동 자세 추적**: 사용자가 스쿼트를 할 때, Vision 프레임워크가 자세를 분석하고 실시간으로 피드백을 제공합니다.
- **다크 모드 UI**: 야간 운동 시에도 눈에 부담이 없는 인터페이스 제공

## 5. 시사점
Mofit iOS 앱은 최신 기술을 통합하여 사용자 경험을 극대화하며, 피트니스 트래킹 앱의 발전 방향을 제시합니다. 특히, AI를 활용한 자세 추적은 개인 맞춤형 피드백 시스템의 가능성을 보여줍니다.
