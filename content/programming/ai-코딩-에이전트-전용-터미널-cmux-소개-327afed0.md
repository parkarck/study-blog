# AI 코딩 에이전트 전용 터미널 cmux 소개

**Category:** programming

**Source:** [https://youtu.be/jGj9yCqN08s](https://youtu.be/jGj9yCqN08s)

**Date:** 2026-03-18

---

cmux는 AI 코딩 에이전트에 최적화된 오픈소스 터미널입니다. 주요 기능: 워크스페이스 관리(프론트엔드/백엔드 분리), 에이전트 알림(작업 완료 시 탭 반짝임), 내장 브라우저(크로미움 기반, 브라우저 자동화 지원), CLI 명령어(거의 모든 작업 제어 가능, 세션 간 컨텍스트 공유). tmux와 비슷하지만 UI 기반으로 접근성이 좋습니다. 단점: 초기 버전이라 불안정, 세션 복구 미지원, 윈도우즈 미지원. AI 코딩 에이전트와 완벽하게 통합된 터미널로 워크스페이스 관리, 알림, 내장 브라우저가 하나의 프로그램에서 통합되어 매우 편리합니다.
