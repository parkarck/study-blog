# Graphify: Instant Knowledge Graph for Claude Code/Antigravity (FREE)

## 1. 핵심 주제
Graphify는 Claude Code와 같은 도구에서 반복적인 데이터 읽기를 줄이고, 프로젝트에 대한 지속적이고 효율적인 이해를 제공하는 지식 그래프 생성 도구입니다.

## 2. 주요 개념 정리
- **Graphify의 정의**: 프로젝트 전체의 구조를 한번 읽고 지식 그래프로 변환하여 Claude가 효율적으로 활용할 수 있게 만드는 도구.
- **작동 원리**: 프로젝트 파일을 분석하고 그 결과를 지식 그래프로 구축하여, Claude가 초기 세션에서 이 그래프를 참조함으로써 불필요한 반복 읽기를 방지.
- **중요한 주장**: Graphify는 프로젝트의 구조적 이해를 돕고, 세션 간의 일관된 데이터 접근을 가능하게 하여 효율성을 높인다.

## 3. 세부 설명
- **첫 번째 단계: 코드 구조 분석**: Python, TypeScript 등 다양한 언어의 파일에서 클래스, 함수, 모듈 등을 파싱하여 구조적 정보를 수집.
- **두 번째 단계: 오디오 및 비디오 분석**: Faster-Whisper를 사용하여 회의 녹음 등을 텍스트로 변환.
- **세 번째 단계: 문서 및 기타 콘텐츠 분석**: 마크다운, PDF 등에서 개념을 추출하고 Claude의 API를 통해 관계를 맺음.
- **지식 그래프 구축**: 각 파일의 개념과 관계를 하나의 그래프로 통합하여 Claude 세션에서 참조.

## 4. 예시 / 적용
- **코드 프로젝트**: 새로운 Claude 세션에서 그래프를 참조하여 필요한 파일만 선택적으로 읽음.
- **연구 논문 및 회의 기록**: 다양한 형식의 문서들을 하나의 연결된 그래프로 관리하여, 효율적인 탐색과 분석 가능.

## 5. 시사점
Graphify는 복잡한 프로젝트의 구조적 이해를 돕고, 반복적인 데이터 접근을 줄여 효율성을 증가시킵니다. 특히, 대규모의 코드 및 문서 프로젝트에서 큰 이점을 제공할 수 있습니다.

## 핵심 정리
- Graphify는 프로젝트를 한번 읽고 지식 그래프로 변환하여 효율적인 데이터 접근을 제공합니다.
- 코드, 오디오/비디오, 문서 등의 데이터를 분석하여 하나의 그래프로 통합합니다.
- Claude는 이 그래프를 참조하여 불필요한 반복 읽기를 줄이고, 더 나은 초기 응답을 제공합니다.
- Graphify는 코드 외에도 연구 및 비즈니스 문서 관리에 유용하게 사용될 수 있습니다.
- 설치 및 업데이트는 간단하며, Git hooks를 통해 변경 사항을 자동으로 반영할 수 있습니다.
- Graphify 사용 시, 긴 세션에서 특히 효율성을 증가시켜 줍니다.

## Timeline

**Core Process:**

1. **프로젝트 초기 이해**: Claude Code에서 새로운 세션을 열고 코드베이스를 이해하기 위해 질문을 합니다. 초기 설정 단계에서 프로젝트의 README 파일을 읽고, 여러 파일을 탐색하여 기초적인 이해를 구축합니다.
   
2. **Graphify 도입**: Graphify는 프로젝트를 한 번만 읽고 모든 연결을 지식 그래프로 구축합니다. Claude는 매 세션마다 이 그래프를 읽어 파일을 반복해서 읽을 필요가 없습니다.

3. **코드 구조 분석**: Graphify는 첫 번째 단계에서 코드 구조를 분석하며, 각 파일의 클래스, 함수, 임포트 등을 파싱합니다. 이는 전적으로 로컬에서 실행되며 외부 API 호출이 필요 없습니다.

4. **오디오 및 비디오 분석**: 두 번째 단계에서는 회의 녹음이나 튜토리얼을 Faster-Whisper를 통해 로컬에서 전사합니다.

5. **기타 콘텐츠 스캔**: 마지막 단계에서는 마크다운, PDF, 이미지 등을 스캔하여 개념과 관계를 추출합니다. 이 과정은 Claude의 API를 한 번만 호출합니다.

6. **지식 그래프 통합**: 모든 데이터는 하나의 그래프로 통합되며, Claude가 세션을 열 때마다 이 그래프를 먼저 읽습니다. 이를 통해 필요한 파일만 선택적으로 읽습니다.

7. **그래프 업데이트 및 활용**: 프로젝트에 변경이 있을 때 그래프를 업데이트하여 최신 상태를 유지합니다. Git 훅을 설치하면 자동으로 그래프가 갱신됩니다.
