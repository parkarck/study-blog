# 27. Tracking Habits

## 1. 핵심 주제
습관 형성과 추적을 위한 시스템을 Amplenote를 활용하여 효과적으로 관리하는 방법.

## 2. 주요 개념 정리

- **습관의 정의**: 일정한 패턴으로 반복적으로 수행하는 행동.
- **작동 원리**: Daily recurrence, task filtering, and completion tracking.
- **중요한 주장**: 습관을 체계적으로 관리하면 일상생활에서의 효율성을 높일 수 있음.
- **근거**: Amplenote의 기능(태그, 필터, 반복 설정 등)을 활용한 습관 관리.
- **시사점**: 디지털 도구를 활용한 체계적인 습관 추적은 목표 달성에 유리함.

## 3. 세부 설명

1. **습관 생성**:
   - 습관을 기록할 노트 작성: 예를 들어, "morning habits".
   - 일일루틴 항목 작성: 반복 주기를 설정해 매일 반복되게 함.
   - 반복 주기 설정: 매일, 이틀마다, 주말 등 다양한 주기로 설정 가능.

2. **습관 추적**:
   - 필터 사용: "morning habits" 노트를 필터링해 특정 습관만 확인.
   - 체크리스트 활용: 매일 습관 체크리스트를 통해 완료된 항목을 체크.
   - 숨겨진 작업 보기: 필요시 'agenda mode'로 전환해 과거 및 미래의 작업 확인.

3. **완료 및 관리**:
   - 완료되지 않은 작업은 'dismiss' 기능으로 관리.
   - 완료한 작업을 나중에 체크해도 문제없음. 수동으로 일정 재조정 가능.
   - 'completion stats' 페이지에서 습관 완료 통계 확인 가능.

## 4. 예시 / 적용

- **예시**: 아침에 일어나서 하는 일련의 습관들(예: 물 마시기, 운동하기)을 "morning habits" 노트에 기록하고 매일 체크리스트를 통해 수행 여부를 확인.
- **완료 통계**: Emojis로 각 습관을 표현하여 시각적으로 직관적인 통계 페이지 구성.

## 5. 시사점

- Amplenote와 같은 디지털 도구를 활용하면 습관 관리가 체계화되며, 이는 일상에서의 생산성을 높임.
- 반복 설정과 필터링 기능은 개인화된 습관 관리에 적합.

## 핵심 정리

- Amplenote의 반복 설정 기능을 통해 습관을 매일, 이틀마다, 주말 등으로 다양하게 관리 가능.
- 필터링 기능을 사용해 특정 습관만 조정하고 추적할 수 있음.
- 완료된 습관은 체크리스트에서 사라지지만, 필요에 따라 'agenda mode'에서 확인 가능.
- 'completion stats'로 습관의 성취도를 시각적으로 파악할 수 있음.
- 디지털 도구는 습관 형성과 추적에 있어 효율성을 제공함.

## Timeline

**Core Process:**

1. **습관 생성**: Amplenote에서 습관을 생성하는 방법을 설명합니다. "아침 습관"이라는 노트를 생성하고, 매일 반복되는 작업을 추가합니다. 반복 주기를 설정하여 매일, 이틀마다, 혹은 주말에만 반복되도록 설정합니다.

2. **태그 및 필터 사용**: 생성된 습관 노트에 태그를 추가하여 조직화합니다. '작업 모드'에서 필터를 사용해 아침 습관 노트만 선택하여 확인할 수 있도록 합니다.

3. **체크리스트 활용**: 매일 아침 체크리스트를 통해 작업을 확인하고 완료된 작업은 목록에서 숨깁니다. 필요 시 '일정 모드'로 전환하여 숨겨진 작업도 볼 수 있습니다.

4. **작업 관리**: 주말 작업을 체크하면 다음 주말로 이동합니다. 작업을 건너뛰거나 잊을 경우에는 '해제' 명령어를 사용하여 관리합니다.

5. **완료 통계 확인**: 완료 통계 페이지에서 가장 긴 연속 기록이나 가장 자주 완료된 습관을 확인합니다. 작업 설명에 이모지를 추가하여 시각적으로 구분합니다.

**Timeline Structure:**

- [00:00] **습관 생성 및 반복 설정**: Amplenote에서 '아침 습관' 노트를 생성하고, 매일 반복되는 작업을 설정합니다. 반복 주기를 설정하여 일일, 이틀 간격, 주말 등에 맞춰 조정합니다. 
- [01:20] **태그 추가 및 필터링**: 습관 노트에 '습관' 태그를 추가하여 조직화하고, 작업 모드에서 필터를 사용해 아침 체크리스트만 볼 수 있도록 설정합니다.
- [02:15] **체크리스트와 일정 모드**: 매일 아침 체크리스트를 활용하여 작업을 완료하고, 완료된 작업은 자동으로 숨깁니다. 필요 시 일정 모드로 숨겨진 작업도 확인할 수 있습니다.
- [03:00] **작업 해제 및 관리**: 주말 작업이 완료되면 다음 주말로 이동합니다. 작업을 건너뛰었을 경우 해제 기능을 사용하여 관리합니다.
- [04:00] **완료 통계 및 시각적 표시**: 완료 통계 페이지를 활용하여 연속 완료 기록을 확인하고, 작업 설명에 이모지를 사용해 시각적 구분을 합니다.

**Raw Insights:**

1. **"반복 주기 설정"**: 매일 반복되는 습관을 설정하고, 반복 주기를 통해 자동화합니다. 이는 일일 효율성을 높이는 데 도움을 줍니다.

2. **"태그와 필터링"**: 노트와 작업을 쉽게 찾을 수 있도록 태그를 사용하고, 필터링 기능을 통해 특정 노트만 집중해서 볼 수 있습니다.

3. **"체크리스트 활용"**: 체크리스트를 통해 매일의 습관을 관리하며, 완료된 작업은 자동으로 숨겨져 깔끔한 인터페이스를 유지합니다.

4. **"작업 해제 기능"**: 작업을 건너뛰거나 잊었을 때 해제 기능을 사용하여 유연하게 관리할 수 있습니다.

5. **"완료 통계"**: 완료 통계 페이지를 통해 연속 기록을 확인하고, 작업 설명에 이모지를 추가하여 시각적 구분을 제공합니다.
