## Description

# k-skill 학습 노트

## 1. 핵심 주제
k-skill은 한국 사용자에게 유용한 다양한 기능을 제공하는 AI 에이전트 모음입니다. 이 문서에서는 k-skill의 기능, 작동 원리, 설치 방법 등을 통해 어떻게 이를 활용할 수 있는지를 설명합니다.

## 2. 주요 개념 정리

### 정의
- **k-skill**: 다양한 일상 업무를 자동화하고 효율적으로 처리할 수 있는 AI 기반 도구 모음.

### 작동 원리
- **에이전트 기반**: 사용자 요청에 따라 다양한 API와 상호작용하여 필요한 정보를 제공하거나 작업을 수행함.
- **프록시 서버**: `k-skill-proxy`를 통해 API 호출을 간편하게 관리.

### 중요한 주장
- AI 에이전트를 통해 반복적이고 귀찮은 작업을 자동화할 수 있다.
- 별도의 클라이언트 API 레이어 없이도 HTTP 요청을 통해 기능을 손쉽게 사용할 수 있음.

## 3. 세부 설명

### 설치 및 시작
1. **설치 방법**: 먼저 k-skill 패키지를 설치합니다.
2. **환경 설정**: `k-skill-setup`을 사용해 필요한 credential 및 환경 변수를 설정합니다.
3. **기능 사용**: 각 기능 문서를 참고하여 입력값 및 제한사항을 확인 후 활용합니다.

### 기능 목록 및 설명
- **SRT/KTX 예매**: 열차 조회, 예약 등을 자동화.
- **카카오톡 Mac CLI**: macOS에서 카카오톡 메시지 관리.
- **서울 지하철 도착정보 조회**: 실시간 도착 정보를 확인.
- **미세먼지 조회**: 현재 위치 기반 미세먼지 농도 확인.
- **한국 법령 검색**: 법령, 조문, 판례 등을 검색.
- **스포츠 경기 결과 조회**: KBO, K리그 경기 결과를 제공.
- **금융 정보 조회**: 토스증권 계좌 및 포트폴리오 정보 조회.
- **생활 정보 조회**: 로또 당첨, HWP 문서 변환, 맛집, 술집 정보 제공.
- **쇼핑 및 배송**: 다이소 상품, 쿠팡 상품, 택배 배송 상태 조회.

## 4. 예시 / 적용
- **열차 예약**: 출발지와 목적지를 입력하여 SRT 또는 KTX 표를 예약.
- **미세먼지 확인**: 현재 위치를 기반으로 실시간 미세먼지 농도를 확인 후 건강 관리.

## 5. 시사점
- **효율성**: 반복적이고 시간이 많이 소요되는 작업을 자동화하여 시간을 절약할 수 있습니다.
- **편리함**: 다양한 서비스를 하나의 플랫폼에서 제공함으로써 사용자 경험을 개선합니다.
- **보안**: 시크릿 및 크리덴셜 관리를 통해 안전하게 서비스를 사용할 수 있습니다.

## 핵심 정리

- k-skill은 다양한 일상 업무를 자동화할 수 있는 AI 기반 도구입니다.
- 프록시 서버를 통해 API 호출을 간편하게 관리합니다.
- 설치는 k-skill 패키지를 설치한 뒤 환경 설정을 완료하는 순서로 진행됩니다.
- SRT/KTX 예매, 미세먼지 조회, 법령 검색 등 다양한 기능을 제공합니다.
- 효율성과 편리함을 제공하며, 보안을 중요시합니다.
