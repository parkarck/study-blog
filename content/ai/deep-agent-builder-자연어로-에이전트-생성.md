## Description

# 자연어로 에이전트 빌더 (Deep Agent Builder) 요약

## 1. 핵심 주제
자연어로 에이전트를 만들 수 있는 AI 플랫폼 "Deep Agent Builder" 소개

## 2. 주요 개념 정리
- **자연어 기반 에이전트 생성:** 사용자 말을 입력하면 7단계 과정을 통해 자동으로 에이전트 생성
- **다양한 LLM 지원:** OpenAI, Anthropic, Google, AWS Bedrock, K-EXAONE, Upstage Solar 100B 등
- **멀티 에이전트:** 메인 에이전트와 서브 에이전트 구조

## 3. 세부 설명

### 3.1 에이전트 생성 과정 (7단계)
1. **사용자 의도 분석:** 객관식 설문조사로 사용자 의도 파악
2. **도구 추천:** 태블릿 서치 등 도구 자동 추천
3. **미들웨어 추천:** 재시도, 모델 제한 등
4. **시스템 프롬프트 작성:** 추출된 의도/도구/미들웨어 기반으로 정교한 프롬프트 생성
5. **설정 저장 및 빌드**
6. **배포 준비**
7. **배포 완료**

### 3.2 주요 기능
- **픽스 에이전트:** 자연어로 프롬프트 수정 가능 (예: "프롬프트를 간략하게 줄여줘")
- **RAG 챗봇:** 문서 업로드 → 자동 RAG 생성 (예: 트럼프 제네시스 미션 관련 행정명령 조회)
- **도구 연동:** 태블릿 서치, 아카이브(논문 검색), MCP 등
- **백그라운드 모드:** 오래 걸리는 검색 작업을 백그라운드에서 처리 후 알림 전송

### 3.3 멀티 에이전트 시스템
- **서브 에이전트 추가/위임:** 웹 검색 에이전트 등 서브 에이전트 연동 가능
- **작업 분리 및 위임:** 복잡한 작업을 여러 에이전트로 분산 처리

### 3.4 데스크톱 앱 (PC 에이전트)
- **로컬 파일 접근:** 사용자 PC의 모든 파일을 안전하게 접근
- **폴더 지정:** 샌드박스 환경에서 특정 폴더만 접근하도록 설정 가능
- **작업 처리:** 파일 정리, 문서 변환, 이미지 생성 등 작업 자동화
- **슬랙 메신저 연동:** 작업 완료 알림을 슬랙으로 전송

## 4. 예시 / 적용
- **예시1 - 웹 검색 에이전트:** "최신 AI 뉴스 알려줘" → 태블릿 서치 도구 사용
- **예시2 - RAG 챗봇:** 트럼프 행정명령 관련 PDF 업로드 → RAG 생성 → 질의 응답
- **예시3 - 멀티 에이전트:** 웹 검색 에이전트(도구) + 문서 검색 에이전트(서브) 협업

## 5. 시사점
- **진입 장벽 해결:** 엠파, 디파이 같은 도구의 코딩 필요성 제거
- **국가대표 LLM 활용:** LG EXAONE, Upstage Solar 등 무료 국가대표 모델 활용
- **생산성 향상:** 반복적인 작업을 백그라운드 모드로 자동화

## 핵심 정리
Deep Agent Builder는 자연어로 에이전트를 생성하는 AI 플랫폼으로, 7단계 자동화 과정을 통해 누구나 쉽게 에이전트를 만들 수 있습니다. 다양한 LLM과 도구를 지원하며, 특히 RAG 챗봇과 백그라운드 모드가 강력한 기능입니다.
