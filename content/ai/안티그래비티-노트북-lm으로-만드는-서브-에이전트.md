## Description

# [안티그래비티 + 노트북 LM으로 만드는 서브 에이전트]

## 1. 핵심 주제
안티그래비티와 노트북 LM을 활용하여 서브 에이전트를 개발하는 방법을 소개합니다. 이를 통해 생산성을 높이고 다양한 자동화 작업을 수행할 수 있습니다.

## 2. 주요 개념 정리
- **안티그래비티**: 자동화 및 실험을 위한 플랫폼으로 다양한 기능과 확장을 지원합니다.
- **노트북 LM**: 노트북 환경에서 AI 모델을 활용하기 위한 도구입니다.
- **서브 에이전트**: 특정 작업을 자동으로 수행하도록 설계된 프로그램입니다.
- **MCP와 스킬스**: 안티그래비티와 노트북 LM을 연결하는 두 가지 방법으로 각각의 기능을 제공합니다.

## 3. 세부 설명

### 안티그래비티와 노트북 LM의 연결
- **MCP 방법**: MCP 서버를 설정하여 두 플랫폼을 연결합니다. 필요한 JSON 파일을 통해 설정을 구성합니다.
- **스킬스 방법**: 기터브의 스킬 저장소에서 스킬을 가져와 연결합니다. 노트북 LM 스킬을 통해 다양한 작업을 수행할 수 있습니다.

### 서브 에이전트의 설계
1. **에이전트 구성**: 
   - 리서치 에이전트: 데이터 수집 및 정리
   - 종합 에이전트: 데이터를 바탕으로 작업 수행
2. **기능 구현**: 
   - 유튜브 영상 수집
   - 노트북 LM에 데이터를 통합
   - 오디오 팟캐스트 생성

## 4. 예시 / 적용
- **팟캐스트 생성 서브 에이전트**:
  - 매일 오전 8시에 특정 유튜버의 24시간 내 영상 데이터를 수집하여 노트북 LM에 입력
  - 수집된 데이터를 오디오 팟캐스트로 변환

## 5. 시사점
- 자동화된 서브 에이전트는 반복적인 작업을 줄여줍니다.
- 안티그래비티와 노트북 LM의 결합은 실무에서의 다양한 응용 가능성을 제시합니다.
- AI와 자동화를 활용한 작업 방식은 앞으로 더욱 확대될 것입니다.

## 핵심 정리
- 안티그래비티와 노트북 LM은 자동화 작업에 강력한 도구입니다.
- MCP와 스킬스를 통해 두 플랫폼을 연결할 수 있습니다.
- 서브 에이전트는 데이터 수집, 정리, 변환을 자동으로 수행합니다.
- 팟캐스트 생성 같은 자동화 작업은 생산성을 크게 향상시킵니다.
- AI와 자동화의 결합은 실무에서 중요한 역할을 합니다.
- 다양한 스킬을 조합하여 맞춤형 에이전트를 구성할 수 있습니다.
