## Description

ChatGPT, Claude 등 LLM 기반 자동화 환경과 Python 개발자를 위해 한국투자증권 Open API를 쉽게 활용할 수 있도록 구성된 샘플 코드 모음. 주식 시세 조회, 주문/정정/취소, 계좌 정보 조회, 실시간 데이터 수신, 백테스팅 기능 지원.
