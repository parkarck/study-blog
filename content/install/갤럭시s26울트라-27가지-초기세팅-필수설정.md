## Description

갤럭시 S26 울트라 초기 설정 가이드

1. 갤럭시 AI 설정: 통화 스크리닝, 알림 하이라이트 켜기
2. 소리 및 진동: 진동 소리 재생, 앱 소리 분리 재생
3. 알림: 알림 내용 숨기기 끄기, 광고/백그라운드 활동 켜기
4. 디스플레이: 다크 모드, 프라이버시 디스플레이(필요시만), 화면 모드 선명하게
5. 배터리: 배터리 보호(85-90% 충전), 미디어 페이지 끄기
6. 홈 화면: 미디어 페이지 끄기, 홈업으로 5x7 배열
7. 보안: 앱 고정 허용, 보안 미동차단 사용 중단 후 자동 켜기
8. 유용 기능: 멀티윈도우 강제, 퍼플렉시티 키보드 설정, 모션 및 제스처 끄기
9. 키보드: 불필요한 문구 추천 끄기, 클립보드 스크린샷 저장 켜기
10. 카메라: 2400만 화소 기본, 고효율 형식, 자동 HDR, 즉시 촬영 켜기
11. 숨겨진 설정: 개발자 옵션 - DW 최소업 조절(주의: 400대 이내)

핵심: 꺼야 하는 것(불필요한 기능)과 켜야 하는 것(필수 기능)을 구분하여 2-3년간 최적화된 사용 가능
